# Triple-C

Objetivo General: Fomentar a la programación en C y su popularidad, dandole un enfoque distinto al habitual, para que sea más atractivo para los que recien entran en el mundo de la programación 

Objetivos Especificos:
1. Creacion de Juego, el cual se tenga una entrada de texto en el que se debe escribir condigo en C para poder jugar y avanzar en el juego.
2. El Juego debe ser divertido y simple, mediante el uso de librerias externas para tener una interfaz atractiva, con la intencion de cambiar la perspectiva de que c es un lenguaje complejo y aburrido.
3. Implementar un sistema de gestión de memoria eficiente en C para el manejo de sprites y entidades, demostrando las ventajas del control manual de recursos.
